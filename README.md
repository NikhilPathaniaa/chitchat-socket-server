# ChitChat Socket.IO Server

This is the Socket.IO server for the ChitChat application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

The server will run on port 3001 by default, or the port specified in the `PORT` environment variable.

## Environment Variables

- `PORT`: The port number for the server (default: 3001)

## Deployment

This server is designed to be deployed to platforms like Render, Railway, or Heroku.
